# Technical SEO Audit

## Description
Comprehensive analysis of website health, indexing issues, speed, and on-page optimization opportunities.

## Scope Includes
- Crawlability analysis
- Site speed performance check
- Keyword gap analysis
- Competitor benchmarking
- PDF Report with recommendations

## Scope Excludes
- Implementation of fixes (See `web/seo_fixes.md`)
- Content writing
- Link building

## Risk Factors
- Large site architecture (>500 pages)
- Access restrictions to GA4/Search Console

## Notes for Agent
This is a **one-time** project price.
If client wants monthly SEO, suggest `seo_monthly_retainer.md`.
