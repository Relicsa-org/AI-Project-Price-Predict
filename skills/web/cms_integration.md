# CMS Integration

## Description
Integration with Headless or Traditional CMS for content management.

## Scope Includes
- CMS Setup (WordPress, Strapi, Sanity, Contentful)
- Content Type Definition
- Admin Panel Configuration
- Frontend CMS Connection
- Content Migration (up to 50 pages)
- Editor Training Session

## Scope Excludes
- Custom plugin development
- Complex workflow automation
- Large content migration (>50 pages)

## Risk Factors
- Legacy content migration issues
- Custom field requirements
- Multi-language content setup

## Notes for Agent
If client mentions "Multi-language", add 15 hours.
If > 50 pages migration, add 5 hours per 50 pages.
WordPress = lower complexity, Strapi/Sanity = medium complexity.
