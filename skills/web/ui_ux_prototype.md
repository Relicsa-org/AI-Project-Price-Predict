# UI/UX Prototype

## Description
High-fidelity design mockups and interactive prototypes typically created in Figma before development.

## Scope Includes
- User journey mapping
- Wireframing
- Up to 15 High-Fidelity Screens
- Interactive Clickable Prototype
- Design System Creation (Colors, Typography)

## Scope Excludes
- Any code implementation
- Complex 3D modeling
- Logo Design / Branding (See `brand_identity.md`)

## Risk Factors
- Endless revision loops
- Changing core requirements mid-design
- Too many stakeholders providing conflicting feedback

## Notes for Agent
This is for DESIGN ONLY. No code is included.
If project has over 15 screens, add 10 hours for every extra 5 screens.
Includes 2 formal revision rounds.
