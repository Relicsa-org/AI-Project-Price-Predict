# Landing Page

## Description
Single-page marketing website with lead capture, responsive design, and basic SEO.

## Scope Includes
- Hero Section with CTA
- Features/Benefits Section
- Contact/Lead Form
- Mobile Responsive Design
- Basic SEO Setup
- Analytics Integration (GA4)

## Scope Excludes
- Multi-page website
- E-commerce functionality
- Custom illustrations/graphics
- Content writing

## Risk Factors
- Client content delivery delays
- Multiple revision rounds
- Complex animations

## Notes for Agent
If client mentions "Multi-page", suggest `ecommerce_store.md` or `saas_dashboard.md`.
If custom graphics needed, add `branding/logo_design.md`.
Includes 2 rounds of revisions.
