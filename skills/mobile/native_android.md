# Native Android App

## Description
Native Android application built with Kotlin/Jetpack Compose for optimal performance and Google ecosystem integration.

## Scope Includes
- Kotlin/Jetpack Compose Development
- 5 Core Screens
- API Integration
- Material Design Guidelines Compliance
- Google Play Store Submission
- Testing on Multiple Devices

## Scope Excludes
- iOS version (separate project)
- Android TV/Wear OS
- Custom ROM compatibility

## Risk Factors
- Android device fragmentation
- Google Play Store policy changes
- Different screen sizes & resolutions

## Notes for Agent
Android only. iOS is separate project.
If client wants Android TV, add 30 hours.
If Wear OS needed, add 40 hours.
Test on minimum 5 device types.
