# Motion Graphics & Animation

## Description
Animated brand assets including logo animation, intro/outro videos, and social media motion graphics.

## Scope Includes
- Logo Animation (5-10 seconds)
- Intro/Outro Video Template
- 3 Social Media Motion Templates
- Multiple Format Exports (MP4, MOV, GIF)
- Background Music Integration

## Scope Excludes
- Full video production
- Live-action filming
- Voice-over recording
- 3D animation

## Risk Factors
- Render time variations
- Platform-specific format requirements
- Revision complexity for animation

## Notes for Agent
This is an ADD-ON to branding projects.
If 3D animation needed, add 50% to base hours.
If > 30 seconds animation, add 20 hours per 30 seconds.
Ensure client provides brand assets beforehand.
